# 🩺 Doctor Appointment System

A MERN Stack application that allows patients to book appointments with doctors and manage schedules easily.  
This project demonstrates **authentication, role-based access, appointments management, and secure APIs**.

---

## 🚀 Features
- 👨‍⚕️ Doctor & Patient roles  
- 🔐 JWT Authentication & Authorization  
- 📅 Appointment booking and management  
- 🗄️ MongoDB database integration  
- ⚡ Express.js REST API  
- 🎨 React.js frontend with modern UI  
- 🔒 Secure environment variables with `.env`

---

## 🛠️ Tech Stack
- **Frontend:** React, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT
